# ポートフォリオサイト

自身の制作物とスキルを紹介するためのポートフォリオサイトです。

## 公開URL
https://aykweb.github.io/portfolio/

## 制作年
2026年

## 使用技術
- HTML5 / CSS3
- JavaScript (Vanilla JS)
- Figma（素材編集）

## 制作のポイント
- **アクセシビリティへの配慮**
  WAI-ARIA（aria-expanded, aria-controls等）やinert属性を用い、スクリーンリーダーやキーボード操作に配慮したモーダルウインドウを実装しました。

- **保守性の高いCSS設計**
  カスタムプロパティを活用し、色の設計や余白を一括管理することで、メンテナンスし易くしています。
